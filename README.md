# dotfiles for macOS

## Requirements
- macOS Catalina (10.15) (or higher)
- `xcode-select --install`
- Sign in to App Store

## Install

`curl -L https://raw.githubusercontent.com/hosht/dotfiles/master/init/install | bash`
